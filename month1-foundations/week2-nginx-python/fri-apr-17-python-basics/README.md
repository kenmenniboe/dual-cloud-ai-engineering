# Python Conditionals & Control Flow

Covered the full conditional logic toolkit in Python: from basic `if` statements through logical operators, with detours into the modulo operator and nested conditions.

---

## Topics Covered

- `if` / `else` — basic decision-making
- `elif` — checking multiple conditions in sequence
- Nested `if` — layered, dependent conditions
- Modulo operator `%` — finding remainders, checking even/odd
- Comparison operators — `==`, `!=`, `>`, `<`, `>=`, `<=`
- Logical operators — `and`, `or`, `not`
- Multiple `if` vs `elif` — knowing when to use which
- Combining `if` with `input()` — making programs interactive

---

## Key Takeaways

- Python evaluates `if–elif–else` chains **top to bottom** and stops at the first `True` condition.
- `=` assigns a value; `==` compares two values — easy to mix up, important to know.
- Nested `if` is for **dependent** conditions; `elif` is for **mutually exclusive** alternatives; multiple `if` is for **independent** checks where more than one can fire.
