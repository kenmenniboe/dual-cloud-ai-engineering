- `if/elif` logic to determine the winner
- `int(input())` to capture and convert user input
- f-strings and escape characters (`\"`, `\n`)

---

## 💡 Key Takeaways

- Lists store multiple items in order, indexed from **0**
- `import random` unlocks Python's randomisation toolbox
- `for` loops visit every item in a list automatically
- List slicing lets you grab a **chunk** of a list with `list[start:stop]`
- List comprehensions build new lists in **one line** of code
- `str()` converts numbers to text; `int()` converts text to numbers
