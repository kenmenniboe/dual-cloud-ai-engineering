# Day 01 — Python Basics (100 Days of Code)

## Overview

Covered foundational Python concepts including:

* `print()` function
* Strings and newline characters
* String concatenation
* `input()` function
* Comments
* Variables and naming conventions
* `len()` function

## Key Examples

### Print and Newline

```python
print("Hello\nKenneth!")
```

**Output:**

```
Hello
Kenneth!
```

### String Concatenation

```python
first_name = "Kenneth"
last_name = "Smith"
full_name = first_name + " " + last_name
print(full_name)
```

**Output:**

```
Kenneth Smith
```

### User Input

```python
name = input("What is your name? ")
print("Hello, " + name + "!")
```

### Length Function

```python
message = "Hello, Kenneth!"
print(len(message))
```

**Output:**

```
15
```

## Takeaways

* Python treats all `input()` values as strings.
* `\n` creates a new line without adding spaces.
* `+` is used for string concatenation.
* `len()` is essential for measuring string/list size.
* Variable naming conventions improve readability and maintainability.
